There are many great ways to contribute!

Shoes is a friendly, inclusive open source project with many ways to get involved. If you are interested in JRuby packaging, web development, graphics, or if you're just starting out, welcome! Here are some ways to get started in the community.

Shoes 4

Shoes 4 is a complete re-write of shoes.

Shoes 4 on GitHub

Shoes 3.3

Shoes 3.3 on GitHub

Shoes

Original Shoes implementation.

Shoes on GitHub

shoesrb.com

Got some web skills? Help improving this website.

shoes.rb on GitHub

Mailing List

There is a new mailing list since the old one fell into the bit bucket! Sadly, you'll have to resubscribe and the archives will not be converted. Discuss Shoes 3.x and Shoes 4, ask for help or pointers or just admire us.

Subscribe here

Facebook

We have a Facebook group. Let people know what Shoes can do. Meet other Shoes people. Post some screen shots of your application. Perhaps a video? Commercial promotion is fine with us as long as it's Shoes relevant. Have Fun!

Facebook

Cmmunity

Contribute and become a part of the team! Many thanks to all who have kept the Shoes spirit alive and helped the project continue.
